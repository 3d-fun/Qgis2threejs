GSI Elevation Tile Provider (DEM Provider)

This DEM provider downloads GSI Tiles (elevation) from the web server of Geospatial Information Authority of Japan, and provides elevation data to Qgis2threejs. The tiled elevation data is based on the popular map tile system known as "XYZ tile", and each tile data is provided in the format of CSV. It covers the area of Japan in zoom levels from 0 to 14. Read and follow Geospatial Information Authority of Japan Website Terms of Use.

Technical information (in Japanese):

  • Specification of elevation tile: http://maps.gsi.go.jp/development/demtile.html
